### 📅 탭 4 · 운행일지
#### 단월 작성
- 차량 / 년도 / 월 선택 후 **🎲 자동 생성**
- 공휴일·일요일(빨강), 토요일(파랑), 출퇴근(초록), 업무(노랑) 색상 구분
- 셀 직접 편집 (주행 전/후/거리 수정)
- **💾 DB 저장** → 엑셀 출력 탭에서 다운로드 가능
- **🖨️ PDF 출력** → html2canvas + jsPDF (화면 색상 그대로 A4 가로)

#### 연간 일괄 생성 (권장)
- 차량 / 년도 / 1월 시작 누적거리 입력
- **🎲 1~12월 일괄 생성 & 저장** → 12개월 자동 생성·저장·누적거리 이월
- 월별 요약 테이블 (시작/종료/총주행/출퇴근/업무/비율) 미리보기
- **📥 연간 엑셀 다운로드** → 연간종합 + 월별 12시트

### 💰 탭 5 · 비용 명세서 ★ 핵심 탭

#### 기본 흐름
```
차량·년도 선택 → 📂 불러오기
  → ⚙️ 고정비용 설정 (자동차세/보험료/할부이자/수선비)
  → ⛽ 주유 기록 입력
  → 비용 셀 직접 입력 (또는 Tab으로 추천값 확정)
  → 💾 저장 → 📥 엑셀 / 🖨️ PDF 출력
```

#### 화면 구성
- **1~12월 전체를 한 화면**에 표시 (월별 섹션 ▲▼ 접기/펼치기)
- 각 월 헤더에 해당 월 소계 실시간 표시
- 하단 **연간 합계 카드** (항목별 색상, 연간 총계)

#### 추천값 시스템 (회색 → 노랑)
| 상태 | 표시 | 의미 |
|---|---|---|
| **회색 placeholder** | 추천값 (미입력) | 시스템 계산값 — 참고만 |
| **노란 배경 숫자** | 입력값 | 사용자가 직접 입력·확정한 값 |
| **회색 이탤릭 "소진"** | 유류비 소진 구간 | 이전 주유분이 아직 소진 중 |

> **Tab 키**: 빈 칸에서 Tab 누르면 추천값이 즉시 노란 입력값으로 확정  
> **직접 타이핑**: 숫자 입력 후 Enter/Tab — 항상 우선 적용

#### ⛽ 유류비 소진 자동 공백 처리
1. 특정 날 유류비 셀에 주유금액(원) 입력
2. 시스템 자동 계산: `(주유금액 ÷ 유가) × 연비 = 주행 가능 km`
3. 이후 날짜의 운행 km 을 순서대로 누적
4. 누적 km ≥ 주행 가능 km 될 때까지 모든 이후 날 유류비 셀 → **"소진" 공백**
5. 소진 완료 다음날부터 → 추천값(회색) 복원
6. 다음 주유 입력 시 해당 구간부터 재적용

```
예시: 50,000원 주유, 연비 12.5km/L, 유가 1,650원/L
  → 50,000 ÷ 1,650 × 12.5 ≈ 378km 소진될 때까지 이후 날 "소진" 표시
```

#### ⛽ 주유 기록 패널
- 날짜 / 금액 / 리터(선택) / 누적km(선택) / 메모 직접 입력
- 마지막 주유 기반 **다음 주유 예상 거리·누적km** 표시
- 연간 총 주행거리 기반 **연간 유류비 추산** 표시

#### ⚙️ 연간 고정비용 설정
| 항목 | 입력 방식 | 추천값 배분 |
|---|---|---|
| 자동차세 | 연간 금액 직접 입력 | ÷12 → 매월 첫 운행일 |
| 보험료 | 연간 금액 직접 입력 | ÷12 → 매월 첫 운행일 |
| 할부이자 | 연간 금액 직접 입력 | ÷12 → 매월 첫 운행일 |
| 수선비 기본 | 월 기본값 직접 입력 | 매월 첫 운행일 |

> 모두 **추천값**으로만 표시 — 실제 입력은 사용자가 직접

#### 📎 영수증 첨부
- 각 비용 행 우측 📎 버튼 클릭
- **파일 선택** (JPG/PNG/PDF) 또는 **📷 카메라 촬영**
- base64로 LocalStorage에 저장
- 썸네일 미리보기, 첨부 개수 배지(숫자) 표시

### 📊 탭 6 · 엑셀 출력

#### 운행기록부 엑셀
- 차량 / 년도 선택 → **📥 운행기록부 엑셀 다운로드**
- 파일명: `운행기록부_등록번호_YYYY년.xlsx`
- 시트 구성:
  - `연간종합`: 월별 총주행/출퇴근/업무/비율 요약
  - `1월`~`12월`: 일별 운행 내역 (색상·이월 적용)

#### 비용명세서 엑셀
- 차량 / 년도 선택 → **📥 비용명세서 엑셀 다운로드**
- 파일명: `비용명세서_등록번호_YYYY년.xlsx`
- 시트 구성:
  - `연간종합비용`: 월별 항목별 합계
  - `1월비용`~`12월비용`: 일별 비용 내역
- 서식: 천단위 콤마 `#,##0` · 헤더 가운데 정렬
- 추천값: 회색 이탤릭 · 입력값: 항목별 색상

## 🗄️ LocalStorage 데이터 구조

| 키 패턴 | 타입 | 내용 |
|---|---|---|
| `driveLog_vehicles` | Array | 차량 목록 |
| `driveLog_clients` | Array | 거래처 목록 |
| `driveLog_settings` | Object | 운행 설정 (출퇴근거리·통행료 등) |
| `driveLog_logs_index` | Array | 운행일지 인덱스 (월별 요약) |
| `driveLog_logs_{vid}_{year}_{mm}` | Object | 월별 운행일지 상세 |
| `driveLog_costdata_{vid}_{year}` | Object | 연간 비용 명세서 (`val`=입력값, `rec`=추천값) |
| `driveLog_annual_{vid}_{year}` | Object | 연간 고정비용 (세금·보험·이자·수선) |
| `driveLog_fuel_{vid}_{year}` | Array | 주유 기록 목록 |
| `driveLog_receipt_{vid}_{y}_{mm}_{key}` | Array | 영수증 파일 (base64) |

## 🎯 빠른 시작 가이드

```
① 차량 관리  → 차량 등록 (연비·유가 입력 필수)
② 거래처 관리 → 방문 거래처 등록 (통행료·주차비 입력)
③ 운행 설정  → 출퇴근 거리·통행료·연간목표 입력 → 저장
④ 운행일지   → 연간 일괄 생성 탭
              → 차량·년도·1월 누적거리 입력
              → [🎲 1~12월 일괄 생성 & 저장] 클릭
⑤ 비용 명세서 → 차량·년도 선택 → [📂 불러오기]
              → [⚙️ 고정비용 설정] 입력 → 저장&재계산
              → [⛽ 주유 기록 패널] 주유할 때마다 기록
              → 셀에 실제 금액 입력 (Tab으로 추천값 확정)
              → [💾 저장] → [📥 엑셀] 또는 [🖨️ PDF]
⑥ 엑셀 출력  → 운행기록부 또는 비용명세서 다운로드
```
